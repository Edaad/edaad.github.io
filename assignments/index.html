Assignment 3: SVD Preprocessing on MNIST with Logistic Regression

In this assignment, I applied Singular Value Decomposition (SVD) to the MNIST dataset as a preprocessing step for dimensionality reduction, followed by training a logistic regression classifier. I experimented with various numbers of SVD components and observed how dimensionality reduction impacted both model accuracy and training time. Additionally, I visualized the top singular vectors to further analyze the effect of SVD on the dataset.

Assignment 4: Latent Semantic Analysis (LSA) Search Engine

This project is a simple search engine web application that implements Latent Semantic Analysis (LSA) to retrieve relevant documents from the 20 Newsgroups dataset. Users can input a search query, and the app returns the top 5 most similar documents based on cosine similarity, along with a bar chart visualizing the similarity scores. The project demonstrates text processing, dimensionality reduction using Singular Value Decomposition (SVD), and real-time document retrieval in a user-friendly interface.

Assignment 5: Predicting Customer Churn Using K-Nearest Neighbors (KNN) Algorithm

This assignment focuses on building a predictive model to identify customers likely to churn from a bank using the **K-Nearest Neighbors (KNN)** algorithm. The task involves implementing KNN from scratch, performing data preprocessing, and tuning hyperparameters like the number of neighbors (`k`) and distance metrics. By evaluating the model's performance using metrics like the ROC AUC score, the goal is to predict churn probabilities for each customer in the test set. This helps banks take proactive steps to improve customer retention.
