Assignment 4: Latent Semantic Analysis (LSA) Search Engine

This project is a simple search engine web application that implements Latent Semantic Analysis (LSA) to retrieve relevant documents from the 20 Newsgroups dataset. Users can input a search query, and the app returns the top 5 most similar documents based on cosine similarity, along with a bar chart visualizing the similarity scores. The project demonstrates text processing, dimensionality reduction using Singular Value Decomposition (SVD), and real-time document retrieval in a user-friendly interface.
